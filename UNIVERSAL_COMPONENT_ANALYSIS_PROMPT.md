## Шаблон ответа (для AI)

При анализе используй следующую структуру ответа:

1. **Сначала анализ** - детальное изучение кода, конфигурации, симуляции
2. **Затем вопросы** - если что-то неясно, спроси перед рекомендациями
3. **Потом рекомендации** - конкретные шаги улучшения
4. **Итоговая оценка** - X/10 с обоснованием

**НЕ пиши код сразу** - сначала анализ, затем спроси нужно ли кодить.
